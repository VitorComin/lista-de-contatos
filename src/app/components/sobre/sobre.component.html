Sobre o Projeto


A aplicação Web consiste em uma lista de contatos, onde é possível armazenar o nome e o número de telefone de cada contato.


Além de habilidades com HTML, CSS, JavaScript e uso de um fakeApi para simular um banco de dados com métodos GET, POST e DELETE (habilidades ja mostradas em outros projetos), neste projeto utilizo Angular, TypeScript e PO UI, ferramentas que publico pela primeira vez em meu GitHub
